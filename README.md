# 使用方法
### 安装依赖：
```
pip install flask requests
```
### 启动服务：
```
python app.py
```
### 访问网页：
本地访问：打开浏览器，输入 http://127.0.0.1:5000
局域网访问：在同一网络下，其他设备输入 http://[服务器IP]:5000（服务器 IP 可在启动日志中查看，如 Running on http://192.168.1.105:5000）
